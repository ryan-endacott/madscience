## Data Discovery: UK BGS Geomagnetic API - May 31, 2025

### Major Upgrade from NOAA:
**Problem with NOAA**: Slow operator queries, potentially unreliable for batch processing

**Solution Found**: British Geological Survey API - official REST endpoint for IGRF-14 data
- **Endpoint**: `https://geomag.bgs.ac.uk/web_service/GMModels/igrf/14/`
- **Format**: JSON responses with exact fields we need
- **Quality**: Official BGS source, citation-worthy
- **Speed**: Automated batch processing vs manual entry
- **Coverage**: IGRF-14 covers 1900-2025, perfect for our 2010 baseline

### API Details:
```
GET https://geomag.bgs.ac.uk/web_service/GMModels/igrf/14/?
    latitude=41.70&longitude=-70.00&
    date=2010-01-01&altitude=0&format=json
```

**Returns exactly what we need**:
- `total-intensity` (F) in nT
- `inclination` (I) in degrees  
- `vertical-intensity` (Z) in nT

## Code Development: BGS Magnetic Harvester - May 31, 2025

### Script Created: `code/bgs_magnetic_harvester.py`

**Purpose**: Automated collection of magnetic field data from BGS API for hypothesis validation
**Target**: 15 global sites (8 stranding hotspots + 7 control sites)
**Method**: 8-point transects around each site for gradient calculation

### Key Features:
- **Official BGS API integration** using IGRF-14 model
- **Rate limiting** (1 second between requests) - respectful of API
- **Robust error handling** for network issues
- **Automatic gradient calculation** from transect measurements
- **CSV output** for analysis pipeline integration
- **Real-time progress reporting** and immediate results

### Site Selection Strategy:
**Stranding Hotspots** (expect positive gradients):
- Farewell Spit, NZ, Cape Cod USA, Tasmania, Norfolk UK
- Golden Bay NZ, Chatham Islands, Prince Edward Island, Orkney Islands

**Control Sites** (expect negative gradients):
- Dutch Wadden Sea, Norwegian Coast, Chilean Coast, Portuguese Coast
- South African Coast, Japanese Coast, Brazilian Coast

### Technical Implementation:
- 8-point transect around each site (50km radius)
- Compares inner vs outer field measurements
- Positive gradient = field increases toward shore (landward-rising)
- Immediate statistical summary of results

**Ready to execute**: Script will process 120 total API calls (15 sites × 8 points) with built-in rate limiting.

## MAJOR DISCOVERY: Hypothesis Replication Failure - May 31, 2025

### BGS Survey Results - CONTRADICTS Original Findings!

**Expected**: Hotspot sites would show positive gradients, control sites negative
**Actual**: Mixed results with NO clear pattern

| Site | Type | BGS Gradient | Original Expectation |
|------|------|--------------|-------------------|
| Farewell Spit, NZ | hotspot | **-1.93 nT/km** | +20.35 nT/km |
| Cape Cod, USA | hotspot | **-1.68 nT/km** | +1.93 nT/km |
| Dutch Wadden Sea | control | **+0.38 nT/km** | -2.55 nT/km |

### Statistical Summary:
- **Hotspot sites with positive gradients**: 2/8 (25%) - Expected: 8/8
- **Control sites with negative gradients**: 2/7 (29%) - Expected: 7/7  
- **Average hotspot gradient**: -0.82 nT/km (negative!) - Expected: positive
- **Average control gradient**: -0.13 nT/km - Expected: strongly negative

## Critical Analysis: What Went Wrong?

### Possible Explanations:
1. **NOAA vs BGS Data Differences**: Different magnetic models giving different results
2. **Methodology Changes**: Our gradient calculation method may have evolved
3. **Coordinate System Issues**: Possible lat/lon or projection differences
4. **Scale Differences**: 50km transects vs original analysis scale
5. **Temporal Factors**: 2010 baseline vs original data dates

### The Scientific Sprint Success:
