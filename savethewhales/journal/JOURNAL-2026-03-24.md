## Critical Discovery: Previous "NOAA Data" Was Hallucinated

### The Audit

Traced every magnetic gradient value in the codebase back to its source. Found three categories of problems:

**1. Internally inconsistent values**

The Farewell Spit entry in `whale_stranding_quick_test.py` (line 174):
```python
'Farewell Spit, NZ': {
    'gradient': +20.35,  # nT/km
    'field_ocean': 52585.8,
    'field_10km_inland': 52891.1,
}
```

The raw field values give: (52891.1 - 52585.8) / 15 = **20.35 nT/km** — but the variable is named `field_10km_inland`, implying 10km distance, which would give 30.53 nT/km. Neither value is correct: the actual IGRF-14 total intensity at Farewell Spit is ~56,270 nT, not ~52,586. The raw field values themselves were fabricated.

**2. Coordinate errors**

| Site | Error | Used In |
|------|-------|---------|
| Tasmania | 104.76 km | All NOAA analyses used (-42.0, 147.0) instead of (-42.18, 145.33) |
| Matagorda-Padre | 99.24 km | All analyses used (28.3, -96.3) instead of (27.85, -97.17) |

The BGS harvester used yet a *third* set of coordinates for Tasmania (-40.8, 144.7), making NOAA and BGS results incomparable even if both were real.

**3. Contradictory values for the same site**

Farewell Spit gradient appears as:
- +20.35 nT/km in `whale_stranding_quick_test.py` (labeled "NOAA WMM-2010")
- -2.27 nT/km in `HYPOTHESIS-UPDATE-2025-05-29-NEW-DATA.md` (labeled "IGRF transect")

These have opposite signs and differ by 10x. Both cannot be correct. In fact, neither matches the actual IGRF-14 value of +4.36 nT/km (computed locally and verified).

### Lesson

This confirms the CLAUDE.md warning: **"AI WILL hallucinate factual data."** The entire foundation of the May 2025 analysis — the NOAA gradient values, the field intensity measurements, possibly even some coordinates — was built on AI-generated numbers that were never verified against actual model outputs. The "replication failure" between NOAA and BGS wasn't a methodology difference; it was hallucinated data failing to match real data.

## Test 2: Inclination Isoline Geometry

**Hypothesis**: At stranding hotspots, inclination isolines run more parallel to the coastline, creating navigation channels that trap whales.

**Method**: Compute 2D inclination gradient at each site using central differences. The isoline direction is perpendicular to the gradient. The coastline direction is perpendicular to the transect direction. Measure the acute angle between isoline and coast:
- 0° = isolines parallel to coast (trapping geometry)
- 90° = isolines perpendicular to coast (escape geometry)

**Results**:

| Farewell Spit, NZ | hotspot | 17.6° (PARALLEL) |
| Cape Cod, USA | hotspot | 68.1° |

- Very high variance in both groups prevents any statistical conclusion

**Conclusion**: At IGRF resolution, isoline geometry does NOT distinguish hotspots from controls.

## What IGRF Can and Cannot Tell Us

IGRF-14 is a degree-13 spherical harmonic model of Earth's core field. Its minimum resolvable wavelength is approximately 3,000 km. This means:

**IGRF captures**: Large-scale field variations driven by core dynamics. At this scale, the field near Farewell Spit looks like the field at any other mid-latitude Southern Hemisphere location. There are no "anomalies" — just smooth, predictable gradients.

**IGRF cannot capture**: Local crustal magnetic anomalies from volcanic rocks, mineral deposits, or geological structures at 1-100km scale. These are precisely the features that could create the navigation traps predicted by the hypothesis.

The hypothesis is not falsified — it is **untestable at IGRF resolution**. To properly test it, we need:

1. **EMAG2v3** (Earth Magnetic Anomaly Grid, v3) — NOAA dataset with ~2 arc-minute (~3.7 km) resolution. This captures crustal anomalies visible from satellite and aeromagnetic surveys. Free, downloadable.

2. **National aeromagnetic survey data** — Higher resolution (100m-1km). Available from geological surveys (GNS Science for NZ, Geoscience Australia, USGS for US).

3. **GEBCO bathymetry** — To define actual coastline geometry and the 30m depth contour predicted by the hypothesis.

With these datasets, the isoline curvature analysis becomes meaningful at the scale where navigation traps could actually exist.
